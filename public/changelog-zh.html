更新日志 (Changelog)

Chat w WebPage 的最新更新和改进。

v1.2.0 2026年1月5日 最新

推理与模型更新

  • 新增: 添加对 DeepSeek Reasoner 和 Gemini 2.0 Flash 思考模型的支持。
  • 新增: 可视化兼容模型的推理步骤（点击展开思维链）。
  • 改进: 增强了数学公式 (LaTeX) 的 Markdown 渲染效果。
  • 修复: 解决了部分自定义 API 端点无法正确保存的问题。
v1.1.0 2025年12月20日

自定义助手与预设

  • 新增: 创建并保存带有特定系统提示词的自定义助手。
  • 新增: 添加"一键总结"预设，便于快速阅读。
  • 改进: 通过 Ollama (localhost:11434) 支持本地模型。
  • 界面: 深色模式现已成为默认体验。
v1.0.0 2025年12月1日

初始发布

  • 发布: Chat w WebPage 允许您与任何活动标签页聊天。
  • 隐私: 采用 BYOK 架构，密钥本地存储。
  • 支持: 初步支持 OpenAI GPT-4 和 Claude 3.5 Sonnet。
  • 核心: 用于文章和文档的内容提取引擎。
