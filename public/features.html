Privacy by Design

We built this extension because we care about privacy. Your browsing data and API keys should belong to you, not us.

  • Local Storage: All keys and history are stored in your browser.
  • Direct Connection: Requests go directly from your browser to the AI provider.
  • Open Source: Our code is public on GitHub for audit.
  • No Middleman: We don't have servers to store your data.
