Changelog

New updates and improvements to Chat w WebPage.

v1.2.0 January 5, 2026 Latest

Reasoning & Models Update

  • New: Added support for DeepSeek Reasoner and Gemini 2.0 Flash thinking models.
  • New: Visualized reasoning steps for compatible models (click to expand chain-of-thought).
  • Improvement: Enhanced Markdown rendering for mathematical formulas (LaTeX).
  • Fix: Resolved an issue where some customized API endpoints were not saving correctly.
v1.1.0 December 20, 2025

Custom Assistants & Presets

  • New: Create and save Custom Assistants with specific system prompts.
  • New: Added "One-Click Summarize" preset for quick reading.
  • Improvement: Added support for local models via Ollama (localhost:11434).
  • UI: Dark mode is now the default experience.
v1.0.0 December 1, 2025

Initial Release

  • Launch: Chat w WebPage allows you to chat with any active tab.
  • Privacy: BYOK architecture with local storage for keys.
  • Support: Initial support for OpenAI GPT-4 and Claude 3.5 Sonnet.
  • Core: Content extraction engine for articles and documentation.
